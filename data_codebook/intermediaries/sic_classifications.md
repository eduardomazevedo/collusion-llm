# sic_classifications.feather

## Description
Complete mapping of SIC (Standard Industrial Classification) codes to their descriptive titles. Includes the full hierarchy from Major Group (2-digit) to Industry (4-digit) levels.

This dataset provides descriptive titles for all SIC classification codes used in the Compustat database. It can be used to map SIC codes in the main dataset to their human-readable names. SIC codes are particularly useful for industry-specific analyses where more granular classification is needed (e.g., chemicals, construction, cement industries) compared to GICS classifications.

## Generated by
`src/pre_query/compustat/download_industry_classifications.py`

## Variables

### sic
SIC classification code. String type, zero-padded to 4 digits. The code length indicates the hierarchy level:
- 2 digits: Major Group (e.g., "28" = Chemicals and Allied Products)
- 3 digits: Industry Group (e.g., "281" = Industrial Inorganic Chemicals)
- 4 digits: Industry (e.g., "2810" = Industrial Inorganic Chemicals, or "3241" = Cement, Hydraulic)

### sicdesc
SIC classification description/title. String type. The human-readable name for the classification code (e.g., "Chemicals and Allied Products", "Cement, Hydraulic").

### level
Classification level (number of significant digits). Integer type. Calculated from the length of the code after removing leading zeros:
- 2: Major Group
- 3: Industry Group
- 4: Industry
