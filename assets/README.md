# Assets Directory

This folder contains configuration files and manually maintained inputs used in the project.

## `gics_sectors.yaml`
Mapping from GICS sector codes to sector names.

## `human_audit_final.xlsx`
Human audit working file with LLM scores, excerpts, and manual audit labels/comments.

## `llm_config.json`
Configuration file for LLM models, capabilities, and token pricing.

## `prompts.json`
Library of prompt versions used for the LLM transcript analysis.
